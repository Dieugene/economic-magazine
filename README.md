# Фронтенд для журнала "Вопросы экономики"

Фронтенд для сайта журнала: клиентская часть и админская для добавления новых номеров и публикаций.

## Цель

Обновление текущего фронтенда, обеспечение удобства для работы с сайтом для людей с гуманитарным образованием.

## Структура проекта

```
.agents/          # AI-агенты для разработки
00_docs/          # Документация и стандарты
01_tasks/         # Задачи разработки
02_src/           # Исходный код
```

## Разработка

Проект использует AI-агентов для организации разработки. Подробности в `DEVELOPER_WORKFLOW.md`.
